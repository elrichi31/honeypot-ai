# Cowrie credential database
# Format: username:unused:password
#   field 2 is currently unused; keep it as x
#   password = *      -> accept ANY password
#   password = !text  -> DENY when password matches "text" (evaluated in order)
#
# Strategy: reject the most common/obvious passwords so the honeypot doesn't
# look trivially breakable, but still let attackers in with less-obvious creds.
# This mimics a server that has basic password policy but was still compromised.
